-------------------------------------------------------------------------------
Scripts
-------------------------------------------------------------------------------

pdstart.sh
	A pd file project manager with backup functionality. Run pdstart.sh without arguments to see usage.

deps.sh
	This script loads a pd patch (argument 1) and puts all dependencies in the directory specified with argument 2.
trim.sh
	Trim a text between delimiters.

_utils.sh
	Useful utilities to work with pd patches

meta.sh
	Gets meta information on a pd patch

absmap.sh
	An abstraction map within a pd patch.

fd_write_run.sh
	Loads the fd_write abstraction to record from terminal

pd2json.py
	A pyhon package to load pd files as json objects. See pd2json-test.py.

config.sh
	creates the Makefile required by pdlibbuilder.
